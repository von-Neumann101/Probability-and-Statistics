## 连续随机变量
**定义**：设连续型随机变量$X$的概率密度为$f(x)$，若积分$\int_{-\infty}^\infty xf(x)dx$绝对收敛，称该积分的值为随机变量$X$的**数学期望**（均值），记为：
$$
\mathbb E[X]=\int_{-\infty}^\infty xf(x)dx
$$
由于期望完全由随机变量$X$的分布确定，所以如果$X$服从某一个分布，那么$\mathbb E[X]$是这个分布的期望

**定理**：设$Y$是随机变量$X$的函数:$Y=g(X)$ ($g$为连续函数)***（重要！）***
1. 如果$X$是离散型随机变量，其分布律为$\Pr\{X=x_k\}=p_k,k=1,2,\ldots$，如果$\sum_{k=1}^\infty g(x_k)p_k$绝对收敛，则有
$$\mathbb E[Y]=\mathbb E[g(X)]=\sum_{k=1}^\infty g(x_k)p_k$$
2. 如果$X$是连续型随机变量，其概率密度为$f(x)$，如果$\int_{-\infty}^\infty g(x)f(x)dx$绝对收敛，则有
$$\mathbb E[Y]=\mathbb E[g(X)]=\int_{-\infty}^\infty g(x)f(x)dx$$
*推广*：设 $Z$ 是随机变量 $X, Y$ 的函数 $Z \! = \! g \left( X, Y \right) \, \left( g \right)$  $g$ 是连续函数），那么，$Z$是一个维随机变量
若二维随机变量 $( X, Y )$ 的概率密度为 $f ( x, y )$ ，则有
$$
E ( Z ) = E { \big [ } g ( X, Y ) { \big ] } = \int _ { - \infty } ^ { \infty } \int _ { - \infty } ^ { \infty } \! g ( x, y ) f ( x, y ) \mathrm { d } x \mathrm { d } y,
$$
若 $( X, Y )$ 为离散型随机变量，其分布律为$P \{ X \! = \! x _ { i } \,, Y \! = \! y _ { j } \} = \! p _ { i j } \,, i \,, j \! = \! 1 \,, 2 \,, \cdots,$ 则有
$$
E ( Z ) = E [ g ( X, Y ) ] = \sum _ { j = 1 } ^ { \infty } \sum _ { i = 1 } ^ { \infty } g ( x _ { i }, y _ { j } ) p _ { i j } \,,
$$
**这里需要上面每个等式右侧的式子绝对收敛**
# 方差
[[Large-Deviations]]
**定义**：设$X$是随机变量，若$\mathbb E\{[X-\mathbb E (x)]^2\}$存在，则称它为$X$的**方差**，记为$D(X)$或$\mathrm{Var}(X)$，引入$\sqrt D(X)$，记为$\sigma (X)$，称为**标准差**或**均方差**

随机变量$X$的方差为$D(X)=E(X^2)-[E(X)]^2$

**标准化**：随机变量$X$具有$E(X)=\mu,\ D(X)=\sigma^2\ne 0$，随机变量$X^*=\dfrac{X-\mu}{\sigma}$的期望为$0$，方差为$1$，称为$X$的**标准化变量**

**性质**：
- 设C是常数，则$D(C)=0$
- 设C是常数，则$D(CX)=C^2D(X),\quad D(X+C)=D(X)$
- 设$X,Y$是随机变量，那么$D(X+Y)=D(X)+D(Y)+2E\{[X-E(X)][Y-E(Y)]\}$. 特别地，当$X,Y$相互独立时，则有$D(X+Y)=D(X)+D(Y)$
- $D(X)=0\Longleftrightarrow \Pr\{X=E(X)\}=1$

**Chebyshev 定理**：$\forall x>0,R$，都有
$$
\Pr(|R-\mathbb E[R]|\ge x)\le\frac{\mathrm{Var}(R)}{x^2}
$$
可以用于证明性质4的必要性
# 协方差及相关系数
**定义**：量$E\{[X-E(X)][Y-E(Y)]\}$称为随机变量$X,Y$的协方差，记为$\mathrm{Cov}(X,Y)$
**定义**：$\rho_{XY}=\dfrac{\mathrm{Cov}(X,Y)}{\sqrt D(X)\sqrt D(Y)}$称为随机变量$X,Y$的相关系数
展开协方差的定义式，得到$\mathrm{Cov}(X,Y)=E(XY)-E(X)E(Y)$
**性质**：
- $\mathrm{Cov}(aX,bY)=ab\mathrm{Cov}(X,Y)$
- $\mathrm{Cov}(X_1+X_2,Y)=\mathrm{Cov}(X_1,Y)+\mathrm{Cov}(X_2,Y)$

$\min_{a,b} E\{[Y-(a+bX)]^2\}=(1-\rho^2_{XY})D(Y)$（$|\rho_{XY}|\le 1$）
所以，当$|\rho_{XY}|$越大，我们称$X,Y$线性相关程度较好，反之则是线性相关程度差。当$\rho_{XY}=0$时称$X,Y$不相关。

之前说过对于二维正态分布随机变量$(X,Y)$，$X,Y$不相关($\rho=0$)等价于$X,Y$相互独立
# 矩、协方差矩阵
设$(X,Y)$是二维随机变量
**定义**：设$X,Y$是随机变量，若
$$
E(X^k),\quad k=1,2,\ldots
$$存在，称其为$X$的 **$k$阶原点矩**，简称$k$阶矩.
若
$$
E\{[X-E(X)]^k\},\quad k=1,2,\ldots
$$
存在，称其为$X$的$k$阶中心矩.
若
$$
E(X^kY^l),\quad k,l=1,2,\ldots
$$
存在，称其为$X,Y$的$k+l$阶混合矩.
若
$$
E\{[X-E(X)]^k[Y-E(Y)]^l\},\quad k,l=1,2,\ldots
$$
存在，称其为$X,Y$的$k+l$阶混合中心矩.

**协方差矩阵**：设$n$维随机变量$(X_1,X_2,\ldots,X_n)$的二阶混合中心矩
$$
c_{ij}=\mathrm{Cov}(X_i,X_j)=E\{[X_i-E(X_i)][X_j-E(X-j)]\},\quad i,j=1,2,\ldots,n
$$
都存在，则称矩阵
$$
\boldsymbol C=
\begin{pmatrix}
c_{11} & c_{12} & \cdots & c_{1n} \\
c_{21} & c_{22} & \cdots & c_{2n} \\
\vdots & \vdots & \ddots & \vdots \\
c_{n1} & c_{n2} & \cdots & c_{nm}
\end{pmatrix}
$$
为$n$维随机变量$(X_1,X_2,\ldots,X_n)$的协方差矩阵。由于$c_{ij}=c_{ji}\quad(i\ne j;i,j=1,2,\ldots,n)$，该矩阵就是一个对称矩阵。

**n维正态随机变量**：
$$
\boldsymbol X=
\begin{pmatrix}
x_1\\
x_2\\
\vdots\\
x_n
\end{pmatrix},
\qquad
\boldsymbol \mu=
\begin{pmatrix}
\mu_1\\
\mu_2\\
\vdots\\
\mu_n
\end{pmatrix}
=
\begin{pmatrix}
E(X_1)\\
E(X_2)\\
\vdots\\
E(X_n)
\end{pmatrix}.
$$
$$
f ( x _ { 1 }, x _ { 2 }, \cdots, x _ { n } ) \! = \! { \frac { 1 } { ( 2 \pi ) ^ { n / 2 } ( \operatorname* { d e t } { \boldsymbol  C } ) ^ { 1 / 2 } } } \! \exp\big \{  - { \frac { 1 } { 2 } } ( { { \boldsymbol  X } } \! - \! { \boldsymbol { \mu } } ) ^ { \intercal } { \boldsymbol  C } ^ { - 1 } ( {  { \boldsymbol  X } } \! - \! { \boldsymbol { \mu } } ) \big\}
$$
其中$\boldsymbol C$是$(X_1,X_2,...,X_n)$的协方差矩阵。
**性质**：
- $n$维正态随机变量$(X_1,X_2,...,X_n)$的每一个分量$X_i,\ i=1,2,\ldots,n$都是正态随机变量；反之，若$X_1,X_2,\ldots,X_n$都是正态随机变量，且互相独立，则$(X_1,X_2,...,X_n)$是$n$维正态随机变量
- $(X_1,X_2,...,X_n)$是$n$维正态随机变量的充要条件是$X_1,X_2,...,X_n$的任意线性组合服从一维正态分布
- 对于$n$维正态随机变量$(X_1,X_2,...,X_n)$的每一个分量$X_i,\ i=1,2,\ldots,n$，他们互相独立等价于两两不相关
