# 二维随机变量
**定义**（二维随机变量）：设 $E$ 是一个随机试验，它的样本空间是 $S = \{ e \}$ ,设 $X  = X (  e )$ 和 $Y = Y ( e )$ 是定义在 $S$上的随机变量，由它们构成的一个向量$(X, Y )$叫做**二维随机向量**或**二维随机变量**

**定义**（联合分布函数）：设（ $X, Y )$ 是二维随机变量，对于任意实数 $x, y$ ，二元函数：$$
F ( x, y ) \! = \! P \{ \left( X \! \leqslant \! x \right) \! \cap ( Y \! \leqslant \! y ) \} { \overset { \mathrm { 记为} }   { \longrightarrow } } P \{X\le x, Y\le y\}
$$称为二元随机变量$(X,Y)$的分布函数，或称为随机变量$X$和$Y$的***联合分布函数***

**性质**：
- $F(x,y)$是变量$x,y$的单调不减函数
- $0\le F(x,y)\le 1$
- 对于任意给定的$y$，$F(-\infty,y)=0$；对于任意给定的$x$，$F(x,-\infty)=0$；$F(-\infty,-\infty)=0$；$F(\infty,\infty)=1$
- $F(x,y)$关于$x,y$右连续

**定义**（二维离散型随机变量）：如果二维随机变量 $( X, Y )$ 全部可能取到的值是**有限对或可列无限多对**，则称 $( X, Y )$ 是**二维离散型随机变量**

**定义**：设二维离散型随机变量 $( X, Y )$ 所有可能取的值为 $( \, x _ { i } \,, y _ { j } \, ) \,, i \,, j = 1 \,, 2 \,, \cdots,$ 记
$P \{ X \! = \! x _ { i } \,, Y \! = \! y _ { j } \} = p _ { i j } \,, i \,, j \! = \! 1 \,, \! 2 \,, \! \cdots,$ 则由概率的定义有
$$
p _ { i j } \geq 0 \,, \quad \sum _ { i = 1 } ^ { \infty } \sum _ { j = 1 } ^ { \infty } p _ { i j } \, = \, 1.
$$
我们称 $P \{ X { = } x _ { i } \,, Y { = } y _ { j } \} = p _ { i j } \,, i \,, j { = } 1 \,, 2 \,, \cdots \, j \,,$ 为二维离散型随机变量 $( X, Y )$ 的分布律，或称为随机变量 $X$ 和 $Y$ 的联合分布律.

**定义**（概率密度）：对于二维随机变量$( X, Y )$的分布函数 $F ( x, y )$ ，如果存在非负可积函数$f ( x, y )$使对于任意 $x, y$ 有
$$
F ( x, y ) = \int _ { - \infty } ^ { y } \int _ { - \infty } ^ { x } f ( u, v ) \, \mathrm { d } u \mathrm { d } v,
$$
则称 $( X, Y )$ 是**二维连续型随机变量**，函数 $f ( x, y )$称为二维连续型随机变量$( X, Y )$ 的**概率密度**，或称为随机变量 $X$ 和 $Y$的***联合概率密度***

**性质**：
- $\textstyle f ( x, y ) \geq 0,$
- 设 $G$ 是 $x O y$ 平面上的区域，点 $( X, Y )$ 落在 $G$ 内的概率为
$$
P \{ ( X, Y ) \in G \} = \iint _ { G } \! \! f ( x, y ) \, \mathrm { d } x \mathrm { d } y.
$$
- 若 $f ( x, y )$ 在点 $( x, y )$ 连续，则有
$$
\frac { \partial ^ { 2 } F ( x, y ) } { \partial x \partial y } \! = \! f ( x, y ).
$$
**近似**：
$$
\operatorname* { l i m } _ { \Delta x \to 0 + \atop \Delta y \to 0 + } \frac { P \{ x < X \leq x + \Delta x, y < Y \leq y + \Delta y \} } { \Delta x \Delta y }=f(x,y)
$$

**定义**（$n$维随机变量）：设 $E$ 是一个随机试验，它的样本空间是 $S = \{ e \}$ ,
设 $X _ { 1 } = X _ { 1 } \left( e \right),X _ { 2 } \! = \! X _ { 2 } \left( e \right), \cdots, X _ { n } \! = \! X _ { n } \left( e \right)$ 是定义在 $\mathit { S }$ 上的随机变量，由它们构成的一个 ${ n }$ 维向量 $( X _ { 1 }, X _ { 2 }, \cdots, X _ { n } )$ 称为${ n }$维随机向量或$n$维随机变量.

**定义**：对于任意 ${ n }$ 个实数 $x _ { 1 } \,, x _ { 2 } \,, \cdots, x _ { n } \,, n$ 元函数
$$
F ( x _ { 1 } \,, x _ { 2 } \,, \cdots, x _ { n } ) \! = \! P \langle \, X _ { 1 } \! \leqslant \! x _ { 1 } \,, X _ { 2 } \! \leqslant \! x _ { 2 } \,, \cdots, X _ { n } \! \leqslant \! x _ { n } \, \rangle
$$
称为 $n$ 维随机变量 $( X _ { 1 }, X _ { 2 }, \cdots, X _ { n } )$ 的**分布函数**或随机变量 $X _ { 1 }, X _ { 2 }, \cdots, X _ { n }$的**联合分布函数**.它具有类似于二维随机变量的分布函数的性质.
