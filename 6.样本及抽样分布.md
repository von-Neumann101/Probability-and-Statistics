# 随机样本
这里我会使用 *采样* 代替所有 **非定义** 段落中的 *观察*

**定义**：将试验的全部可能的观察值称为**总体**，每一个可能观察值称为**个体**。总体中所包含的个体的个数称为总体的**容量**。容量为有限的称为**有限总体**，容量为无限的称为**无限总体**

**注意**：随机变量$X$是一个**函数**，所以一个**总体**对应一个**随机变量**$X$，每一个个体都是他的一个**取值**。
从总体抽取一个个体，就是对总体$X$进行一次采样——对总体$X$进行$n$次重复、**独立**的采样

**定理**：当个体的总数$N$远大于样本的容量$n$时，可以将**不放回抽样近似为放回抽样处理**

**定义**：设 $X$ 是具有分布函数 $F$ 的随机变量，若 $X _ { 1 }, X _ { 2 }, \cdots, X _ { n }$是具有同一分布函数 $F$ 的、相互独立的随机变量，则称 $X _ { 1 }, X _ { 2 }, \cdots, X _ { n }$ 为从分布函数 $F$ （或总体 $F$、或总体 $X$ ）得到的**容量为 $n$ 的简单随机样本**，简称**样本**，它们的观察值$x _ { 1 }, x _ { 2 }, \cdots, x _ { n }$ 称为**样本值**，又称为 $X$ 的 $n$个**独立的观察值**。
也可以将样本看成是一个随机向量，写成 $( \, X _ { 1 } \,, X _ { 2 } \,, \cdots, X _ { n } \, )$ ，此时样本值相应地写成高维向量的形式：$( x _ { 1 }, x _ { 2 }, \cdots, x _ { n } )$、$( \, y _ { 1 } \,, y _ { 2 } \,, \cdots, y _ { n } \, )$，作为样本向量相应的样本值向量

显然，由独立性可得$(X_1,X_2,...,X_n)$的分布函数和概率密度（如果存在）：
$$
\begin{align*}
&F^*(x_1,x_2,...,x_n)=\prod_{i=1}^nF(x_i)\\
&f^*(x_1,x_2,...,x_n)=\prod_{i=1}^nf(x_i)
\end{align*}
$$
