# 大数定律
**弱大数定律**（辛钦大数定理）：设$X_1,X_2,...$相互独立，服从同一个分布的随机变量序列，具有数学期望$E(X_k)=\mu\ (k=1,2,...)$，对于任意$\varepsilon>0$，有（这里意思是分布的均值为$\mu$）
$$
\lim_{n\to\infty}P\left\{\left|\frac{1}{n}\sum_{k=1}^nX_k-\mu\right|<\varepsilon\right\}=1
$$
Proof.
切比雪夫不等式

**定义**：设$X_1,X_2,...,X_n,...$是一个随机变量序列，$a$是一个
常数。若对于任意$\varepsilon>0$，有
$$
\lim_{n\to\infty}P\{|X_n-a|<\varepsilon\}=1
$$
则称序列$X_1,X_2,...,X_n,...$依概率收敛于$a$，记为
$$X_n \stackrel{P}{\longrightarrow} a$$
**定理**：设$X_n \stackrel{P}{\longrightarrow} a,Y_n \stackrel{P}{\longrightarrow} b$，又设函数$g(x,y)$在$(a,b)$连续，则
$$g(X_n,Y_n) \stackrel{P}{\longrightarrow} g(a,b)$$

**伯努利大数定理**：设$f_A$是$n$次独立重复试验中时间$A$发生的次数，$p$是事件$A$在每次试验中发生的概率，则对于任意$\varepsilon>0$，有
$$
\lim_{n\to\infty}P\left\{\left|\frac{f_A}{n}-p\right|<\varepsilon\right\}=1
$$
# 中心极限定理
**定理1**：（独立同分布的中心极限定理）设随机变量 $X _ { 1 } \,, X _ { 2 } \,, \cdots, X _ { n } \,, \cdots$ **相互独立，服从同一分布**，且具有数学期望和方差： $E ( X _ { k } ) \! = \!  { \mu }, \! D \ ( X _ { k } ) \! = \! { \sigma } ^ { 2 } \! > \! 0 \ ( k \! = \! 1 \,,2, \cdots )$ ，则**随机变量之和** $\sum _ { k = 1 } ^ { n } X _ { k }$ 的标准化变量
$$
Y _ { n } = { \frac { \sum _{ k = 1 } ^ n  X _ { k } - E \Big ( \sum _ { k = 1 } ^ { n } X _ { k } \Big ) } { \sqrt { D { \Big ( } \sum _ { k = 1 } ^ { n } X _ { k } { \Big ) } } } } = { \frac { \sum _ { k = 1 } ^ { n } X _ { k } - n\mu } { \sqrt { n } \sigma } }
$$
的分布函数$F_n(x)$对于任意$x$满足
$$
\lim_{n\to\infty}F_n(x)=\lim_{n\to\infty} P\left\{\dfrac{\sum_{k=1}^{n}X_k-n\mu}{\sqrt n\sigma}\le x\right\}=\int_{-\infty}^x\frac1{\sqrt{2\pi}}e^{-t^2/2}dt=\Phi(x)
$$
即
$$
\dfrac{\sum_{k=1}^{n}X_k-n\mu}{\sqrt n\sigma}\sim\mathcal N(0,1)
$$
或者说均值为$\mu$，方差为$\sigma^2$的独立同分布随机变量$X_1,X_2,...,X_n$的算术平均，在$n$充分大时近似地服从均值为$\mu$，方差为$\sigma^2/n$的**正态分布**

---
**定理2**(Lyapunov)：设随机变量 $X _ { 1 } \,, X _ { 2 } \,, \cdots, X _ { n } \,, \cdots$ **相互独立**，他们具有数学期望和方差
$$
E(X_k)=\mu_k,\quad D(X_k)=\sigma_k^2,\quad k=1,2,...,
$$
记$B_n=\sum_{k=1}^n\sigma_k^2$，若存在正数$\delta$，使得当$n\to\infty$时，
$$
\dfrac1{B_n^{2+\delta}}\sum_{k=1}^nE\{|X_k-\mu_k|^{2+\delta}\}\to 0,
$$
则**随机变量之和** $\sum _ { k = 1 } ^ { n } X _ { k }$ 的标准化变量
$$
Z _ { n } = { \frac { \sum _{ k = 1 } ^ n  X _ { k } - E \Big ( \sum _ { k = 1 } ^ { n } X _ { k } \Big ) } { \sqrt { D { \Big ( } \sum _ { k = 1 } ^ { n } X _ { k } { \Big ) } } } } = { \frac { \sum _ { k = 1 } ^ { n } X _ { k } - \sum _ { k = 1 } ^ { n }\mu_k } { B_n } }
$$
的分布函数$F_n(x)$对于任意$x$，满足
$$
\begin{align*}
\lim_{n\to \infty}F_n(x)&=\lim_{n\to \infty}P\left\{\dfrac{\sum_{k=1}^nX_k-\sum_{k=1}^n\mu_k}{B_n}\le x\right\}\\
&=\int_{-\infty}^x\frac1{\sqrt{2\pi}}e^{-t^2/2}dt=\Phi(x)
\end{align*}
$$
无论各个随机变量$X_k(k=1,2,...)$服从什么分布，只要满足定理的条件，那么他们的和在$n$很大的时候，近似服从正态分布

---
**定理3**(De Moivre-Laplace)：设随机变量$\eta_n(n=1,2,...)\sim B(n,p)$，则对于任意$x$，有
$$
\lim_{n\to \infty}P\left\{\dfrac{\eta_n-np}{\sqrt{np(1-p)}}\le x\right\}=\int_{-\infty}^x\frac1{\sqrt{2\pi}}e^{-t^2/2}dt=\Phi(x)
$$
将$\eta_n$分解为$n$个独立的，服从0-1分布的随机变量之和，使用**定理1**即证
