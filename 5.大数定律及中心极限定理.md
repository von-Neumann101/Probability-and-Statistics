# 大数定律
**弱大数定律**（辛钦大数定理）：设$X_1,X_2,...$相互独立，服从同一个分布的随机变量序列，具有数学期望$E(X_k)=\mu\ (k=1,2,...)$，对于任意$\varepsilon>0$，有（这里意思是分布的均值为$\mu$）
$$
\lim_{n\to\infty}P\left\{\left|\frac{1}{n}\sum_{k=1}^nX_k-\mu\right|<\varepsilon\right\}=0
$$
Proof.
切比雪夫不等式

**定义**：设$X_1,X_2,...,X_n,...$是一个随机变量序列，$a$是一个常数。若对于任意$\varepsilon>0$，有
$$
\lim_{n\to\infty}\{|X_n-a|<\varepsilon\}=1
$$
则称序列$X_1,X_2,...,X_n,...$依概率收敛于$a$，记为
$$X_n \stackrel{P}{\longrightarrow} a$$
**定理**：设$X_n \stackrel{P}{\longrightarrow} a,Y_n \stackrel{P}{\longrightarrow} b$，又设函数$g(x,y)$在$(a,b)$连续，则
$$g(X_n,Y_n) \stackrel{P}{\longrightarrow} g(a,b)$$

**伯努利大数定理**：设$f_A$是$n$次独立重复试验中时间$A$发生的次数，$p$是事件$A$在每次试验中发生的概率，则对于任意$\varepsilon>0$，有
$$
\lim_{n\to\infty}P\left\{\left|\frac{f_A}{n}-p\right|<\varepsilon\right\}=1
$$
# 中心极限定理
**定理**：（独立同分布的中心极限定理）设随机变量 $X _ { 1 } \,, X _ { 2 } \,, \cdots, X _ { n } \,, \cdots$ 相互独立，服从同一分布，且具有数学期望和方差： $E ( X _ { k } ) \! = \!  { \mu }, \! D \ ( X _ { k } ) \! = \! { \sigma } ^ { 2 } \! > \! 0 \ ( k \! = \! 1 \,,2, \cdots )$ ，则随机变量之和 $\sum _ { k = 1 } ^ { n } X _ { k }$ 的标准化变量

$$
Y _ { n } = { \frac { \sum _{ k = 1 } ^ n  X _ { k } - E \Big ( \sum _ { k = 1 } ^ { n } X _ { k } \Big ) } { \sqrt { D { \Big ( } \sum _ { k = 1 } ^ { n } X _ { k } { \Big ) } } } } = { \frac { \sum _ { k = 1 } ^ { n } X _ { k } - \eta \mu } { \sqrt { n } \sigma } }
$$
的分布函数$F_n(x)$对于任意$x$满足
$$
\lim_{n\to\infty}F_n(x)=\lim_{n\to\infty} P\left\{\dfrac{\sum_{k=1}^{n}X_k-n\mu}{\sqrt n\sigma}\le x\right\}=\int_{-\infty}^x\frac1{\sqrt{2\pi}}e^{-t^2/2}dt=\Phi(x)
$$
即
$$
\dfrac{\sum_{k=1}^{n}X_k-n\mu}{\sqrt n\sigma}\sim\mathcal N(0,1)
$$
